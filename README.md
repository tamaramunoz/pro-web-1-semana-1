# Control Semana 1 - Programación Web I

## Descripción del Proyecto:
Este repositorio contiene el piloto de una página web básica para un restaurante de gastronomía chilena en Santiago. El objetivo es aplicar los fundamentos de HTML5 aprendidos durante la semana, incluyendo estructura básica, secciones semánticas y etiquetas de formato.

## Funcionalidades y Estructura:
* **Estructura Semántica**: Implementación de etiquetas raíz, cabecera (`head`) y cuerpo (`body`).
* **Secciones de Contenido**: Organización mediante etiquetas `<header>`, `<nav>`, `<main>`, `<section>`, `<article>` y `<footer>`.
* **Formato de Texto**: Uso de etiquetas de énfasis (`<strong>`, `<em>`) y formato de color (`<font>`) según los requerimientos de la semana.
* **Contacto**: Información del local estructurada con la etiqueta `<address>`.

## Herramientas Utilizadas:
* **Editor de Código**: Visual Studio Code.
* **Lenguaje**: HTML5.
* **Control de Versiones**: Git y GitHub.
